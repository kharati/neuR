## SetUp

To **install** this package (in R):

    #if devtools is not installed yet: 
    # install.packages("devtools") 
    library(devtools)
    install_github("livioivil/neuR")
    library(neuR)
    
    #This may be also useful
    library(parallel)
    options(mc.cores=min(detectCores()-1,30))
